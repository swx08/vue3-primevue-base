# Vue 3 + Vite + Primevue

## 介绍
> 基础模板，不包含动态路由和权限控制，仅包含基础功能

## 1. 安装依赖
```bash
npm install
```

## 2. 运行项目
```bash
npm run dev
```

## 3. 打包项目
```bash
npm run build
```

## 集成vue-sidebar-menu菜单栏组件

>github地址：https://github.com/yaminncco/vue-sidebar-menu

>官网地址：https://yaminncco.github.io/vue-sidebar-menu/#/

```bash
# 安装依赖
npm i vue-sidebar-menu --save
```

具体使用方法参考官网
